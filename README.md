# chaechae

전북우리사이 청년 자원봉사단체 웹사이트를 직접 개발·운영하며 만든 프로젝트입니다.

## 🔧 Tech Stack
- HTML, CSS, JavaScript
- PHP, MySQL
- GNUBoard

## ✨ Features
- 반응형 웹 퍼블리싱
- 게시판 및 관리자 페이지(그누보드 연동)
- 봉사활동 기록 관리 CRUD
- 실제 운영 사이트 유지보수 경험

## 📁 Project Structure
- /style : 공통 및 반응형 CSS
- /includes : header / footer
- /gnuboard : 게시판 및 회원 기능
- /page, /post : 콘텐츠 페이지

## 📌 What I Learned
- 유지보수 가능한 CSS 구조 설계
- PHP–MySQL 연동을 통한 데이터 처리
- 실서비스 환경에서의 오류 수정 및 개선
