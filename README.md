# chaechae
전북우리사이 웹사이트 개발/운영을 하며 만든 프로젝트입니다.

## Tech
HTML / CSS / JavaScript / PHP / MySQL / GNUBoard

## Features
- 반응형 UI
- 게시판/관리자 기능(그누보드 연동)
- 봉사활동 기록 관리 CRUD(예: 추가/삭제/조회)

## Structure
- /style : CSS
- /includes : 공통 헤더/푸터
- /gnuboard : 그누보드
